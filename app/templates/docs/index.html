登入 / 帳號

登入步驟

  1. 開啟 /finance/ 或 /finance/quick/，右上方點擊「登入」。
  2. 輸入帳號與密碼，勾選「記住我 30 天」後點擊「登入」。
  3. 登入後 JWT Token 存於瀏覽器 localStorage，有效期間內重新整理不需再次登入。
  4. 點擊右上角使用者名稱 → 「登出」可清除 Token 並返回登入狀態。
記住我 30 天：勾選後 Token 有效期延長至 30 天；未勾選預設 8 小時後過期（依伺服器設定）。

自助註冊

  1. 登入頁點擊「還沒有帳號？立即註冊 →」切換至註冊表單。
  2. 填入帳號（3–32 字元，英數字／底線／連字號）、密碼（至少 6 字元）與顯示名稱（選填）。
  3. 點擊「建立帳號」，成功後自動登入並跳轉至主頁。
管理員注意：自助註冊功能預設為關閉。若需開放，請在伺服器 .env 設定 REGISTER_ENABLED=true 後重啟服務。關閉時，僅管理員可在系統後台 /admin/ 建立帳號。

修改密碼 / 個人資料

  1. 登入後台後，點擊右上角使用者名稱 → 「個人設定」。
  2. 可修改顯示名稱、Email，或點擊「修改密碼」輸入舊密碼與新密碼。
  3. 修改密碼後需重新登入。
首次登入：系統預設帳號為 admin / admin，請立即修改密碼。

收支記錄

新增記錄

  1. 切換至後台「收支」分頁，點擊右上角「＋ 新增」按鈕。
  2. 選擇類型（收入 / 支出）、填入金額、日期、分類（可選）與說明。
  3. 點擊「儲存」，新記錄出現在列表頂端。

篩選與搜尋

在列表上方使用以下篩選條件：

條件說明
日期範圍設定開始日期與結束日期
類型全部 / 收入 / 支出
分類下拉選擇特定分類
關鍵字搜尋說明欄位

編輯 / 刪除

  1. 點擊記錄列末端的「✏️ 編輯」圖示開啟編輯表單，修改後儲存。
  2. 點擊「🗑 刪除」圖示，確認後永久刪除該筆記錄。

分類管理

系統啟動時自動建立 16 個預設分類（10 支出 + 6 收入），可自行新增或修改。

  1. 切換至後台「分類」分頁。
  2. 點擊「＋ 新增分類」，填入名稱、類型（收入 / 支出）、顏色與圖示（Bootstrap Icons class）。
  3. 點擊分類右側「✏️」可修改，「🗑」可刪除（有關聯記錄的分類無法刪除）。
圖示可至 Bootstrap Icons 網站查詢，複製 class 名稱（如 bi-house）填入圖示欄位。

定期交易

設定定期自動產生的收支項目，例如每月薪水、房租、訂閱費。

  1. 切換至後台「定期交易」分頁，點擊「＋ 新增」。
  2. 填入名稱、類型、金額、分類、頻率（每月 / 每年）與起始日期。
  3. 設定後系統依照頻率自動在對應月份顯示待執行項目。
  4. 每月點擊「執行」將定期項目轉為正式收支記錄。

股票交易

新增股票記錄

  1. 切換至後台「股票」分頁，點擊「＋ 新增」。
  2. 填入股票代號（自動轉大寫）、市場（台股 TW / 美股 US / 港股 HK）。
  3. 選擇動作：買入、賣出 或 股利。
  4. 填入股數、單價、手續費（選填）後儲存。
動作說明
買入（buy）增加持倉股數，計入平均成本
賣出（sell）減少持倉股數，計算已實現損益
股利（dividend）記錄現金股利，不影響持倉股數

持倉與損益

持倉彙總

「股票」分頁上方顯示各持倉股票的持有股數與平均成本。

欄位計算方式
持有股數買入總股數 − 賣出總股數
平均成本買入總金額 ÷ 買入總股數
已實現損益賣出金額 − 對應買入成本（含手續費）
持倉為 0 的股票不顯示在持倉列表，但交易記錄仍保留。

預算管理

  1. 切換至後台「預算」分頁，選擇年份與月份。
  2. 針對各支出分類輸入當月預算金額，點擊「儲存」（系統採 UPSERT，重複儲存不會產生重複資料）。
  3. 「執行率」區塊即時顯示各分類已用金額 / 預算金額與百分比。
使用率顯示
< 80%綠色進度條，正常
80% – 100%黃色進度條，接近上限
> 100%紅色進度條，已超支

報表分析

切換至後台「報表」分頁，提供以下視覺化圖表：

報表說明
年度收支趨勢長條圖，比較全年各月份收入與支出金額
月份支出分類圓餅圖，顯示當月各支出分類佔比
月報摘要本月收入、支出、結餘、各分類小計
儀表板摘要本月概況、預算執行率快覽、持倉數、最近 5 筆交易

資料匯出

  1. 切換至「報表」分頁，點擊右上角「匯出」按鈕。
  2. 選擇資料類型（收支記錄 / 股票記錄）與格式（CSV / Excel）。
  3. 可指定日期範圍，點擊「下載」後瀏覽器自動下載檔案。
CSV 格式採 UTF-8 BOM 編碼，可直接以 Microsoft Excel 開啟並正確顯示中文。

手機快速記帳

開啟 /finance/quick/，提供行動裝置最佳化的大按鈕輸入介面。 上方模式列切換「💰 收支」與「📈 股票」兩種模式。

💰 收支模式

  1. 點擊上方模式列的「💰 收支」（預設為此模式）。
  2. 選擇類型（支出 / 收入），點擊分類圖示。
  3. 使用數字鍵盤輸入金額（支援小數點）。
  4. 填入說明（選填），確認日期後點擊「記帳」送出。
  5. 成功後金額與說明自動清空，可繼續下一筆記錄。
近期記錄顯示於頁面下方，可點擊切換「收支」與「股票」兩個紀錄分頁。

📈 股票快速紀錄

  1. 點擊上方模式列的「📈 股票」切換至股票模式。
  2. 輸入股票代號（自動轉大寫，例如 2330 或 AAPL）。
  3. 選擇市場（台股 / 美股）與動作（買入 / 賣出 / 股利）。
  4. 點擊欄位標籤切換目前輸入目標：
    • 股數：輸入買賣股數（股利模式下輸入股利金額）
    • 價格：輸入每股單價
    • 手續費：輸入手續費（選填）
  5. 底部自動計算並顯示總金額（股數 × 價格 ＋ 手續費）。
  6. 點擊「記錄股票」送出，成功後欄位自動清空。
股利模式：「股數」欄位輸入股利金額，「價格」欄位無需填入，系統會以股利金額為準記錄。

快速記帳登入

  1. 首次使用時頁面底部顯示登入 / 註冊區塊。
  2. 輸入帳號與密碼（或切換至「註冊」分頁建立新帳號）後登入。
  3. 登入後記帳操作會自動與你的帳號關聯。

角色與權限

角色可存取功能
admin 全部功能：理財後台、手機快速記帳、系統後台（/admin/）、使用者管理、操作紀錄
user 理財後台（/finance/）與手機快速記帳（/finance/quick/）；各帳號資料互相隔離
各使用者的收支、股票、預算與分類資料完全隔離，admin 可在系統後台查看所有使用者。

使用者管理

僅 admin 可存取 /admin/ 系統後台進行使用者管理。

新增使用者

  1. 進入 /admin/，切換至「使用者管理」分頁。
  2. 點擊「＋ 新增使用者」，填入帳號、密碼、顯示名稱與角色（admin / user）。
  3. 點擊「建立」，新帳號立即可登入，系統自動初始化預設財務分類。

停用 / 啟用帳號

  1. 在使用者列表點擊帳號旁的「⚙️ 操作」。
  2. 選擇「停用」可禁止該帳號登入（帳號資料保留）；選擇「啟用」恢復登入權限。

重設密碼

  1. 在使用者列表找到目標帳號，點擊「重設密碼」。
  2. 輸入新密碼後確認，下次登入時使用者需使用新密碼。
